JavaScript Array 对象

2019-12-09 JavaScript JavaScript 0 评论 字数统计:2710字

结合实际例子与使用方式介绍Array对象的方法。

JS中 Array 对象的方法使用详解:

1.常用的基本方法

  • concat() (合并数组)连接两个或个多的数组，并返回结果。传入的可以是具体值、也可以是数组对象。 该方法不改变原数组

    let arr1 = [1, 2, 3];
let arr2 = [4, 5];

arr1.concat(arr2, 6);   // [1, 2, 3, 4, 5, 6]

  • join() 把数组的所有元素放入一个字符串。元素通过指定的分隔符进行分隔，如果省略该参数，则默认使用逗号作为分隔符。

    let arr = [1, 2, 3];

arr.join('|');  // "1|2|3"

  • pop() 删除并返回数组的最后一个元素，把数组长度减 1。如果数组已经为空，则 pop() 不改变数组，并返回 undefined 值。

    let arr = [1, 2, 3];

arr.pop();  // 3

  • push() 向数组的末尾添加一个或多个元素，返回新的长度。
    push() 方法和 pop() 方法使用数组提供的先进后出栈的功能。

    let arr = ["Bob", "John"];

arr.push("James");      // 3
console.log(arr);       // ["Bob", "John", "James"]

  • shift() 删除并返回数组的第一个元素。将已经存在的元素顺次往前移。

    let arr = [1, 2, 3];

arr.shift();  // 1

  • unshift() 向数组的开头添加一个或多个元素，返回新的长度。将参数插入数组的头部，将已经存在的元素顺次往后移。
    unshift() 方法无法在 Internet Explorer 中正确地工作！

    let arr = ["Bob", "John"];

arr.unshift("James");   // 3
console.log(arr);       // ["James", "Bob", "John"]

  • reverse() 颠倒数组中元素的顺序。
    不常用

  • slice() (裁剪数组)根据下标返回数组中选定的元素。包含start和end两个参数，可以为负数(从数组尾部开始算起)。如果 end 未被规定，会选取从 start 到数组结尾的所有元素。

    let arr = ["George", "John", "Thomas", "James", "Adrew"];

arr.slice(2);       // ["Thomas", "James", "Adrew"]
arr.slice(2, 4);    // ["Thomas", "James"]

  • splice() (拼接数组)向数组指定位置删除或添加元素，返回被删除项目的新数组。删除的元素个数可以跟替换的元素个数不相等。

    let arr = ["Bob", "John", "James"];
// 1.添加1个元素
arr.splice(1, 0, "William");    // [] 如果没有删除元素则返回空
console.log(arr);               // ["Bob", "William", "John", "James"]
// 2.删除1个元素
arr = ["Bob", "John", "James"];

arr.splice(1, 1);               // ["John"]
console.log(arr);               // ["Bob", "James"]
// 3.删除2个元素 添加1个元素来替换
arr = ["Bob", "John", "James"];

arr.splice(1, 2, "William");    // ["John", "James"]
console.log(arr);               // ["Bob", "William"]

  • sort() 对数组的元素进行排序。传入的必须是函数，如果没有传入参数，将按照内置算法对元素进行排序。 该方法在原数组上进行排序
    [内置算法]: JS中sort实现的算法是O(n^2)的冒泡排序，并会将每一项通过toString()转型，即将每一项转化为ASCII进行比较。
    如果想按照其他标准进行排序，就需要提供比较函数，该函数应该具有两个参数 a 和 b，其返回值如下：

    • 若 a 小于 b，则返回一个小于0的值。在排序后的数组中 a 将出现在 b 之前
    • 若 a 等于 b，则返回0。
    • 若 a 大于 b，则返回一个大于0的值。
    let arr = [10, 5, 40, 25, 100, 1];

// 内置算法会将数组中的每一项转化为ASCII进行比较，所以排序顺序有误。
arr.sort();     // [1, 10, 25, 40, 5]

// 常见的比较函数有：
// 1.按照数值大小升序排序
arr.sort(function(a,b){
return a - b;
})；            // [1, 5, 10, 25, 40, 100] 
// 2.按照数值大小降序排序
arr.sort(function(a,b){
return b - a;
});             // [100, 40, 25, 10, 5, 1] 
// 3.按照对象的某一属性值排序
arr = [
{name: "Bob", age: 13}, 
{name: "John", age: 22}, 
{name: "James", age: 7}
];

arr.sort(function(a, b){
var value1 = a.age;
var value2 = b.age;
return value1 - value2;
});
// 4.打乱数组
arr.sort(function(a,b){
return Math.random() - 0.5;
})

  • toString() 把数组转换为(逗号分隔的)字符串。返回值与没有参数的 join() 方法返回的字符串相同。

    let arr = [1, 2, 3];

arr.toString();     // "1, 2, 3"

2.迭代方法

  • every() 用于检测数组所有元素是否都符合某个指定函数的条件。 该方法不改变原始数组
    若使用空数组，此方法在一切情况下都返回 true。

    every() 方法使用指定函数检测数组中的所有元素：

    • 该函数接收3个参数，当前值_currentVal(必须)，当前值索引_index(可选)，当前数组_array(可选)。
    • 如果数组中检测到有一个元素不满足，则整个表达式返回 false ，且剩余的元素不会再进行检测。
    • 如果所有元素都满足条件，则返回 true。
    // 检测数组所有元素是否都大于10
function compare(val){
return val > 10;
}

let arr1 = [1, 11, 2, 22];
let arr2 = [20, 30, 40, 50];

arr1.every(compare);        // false
arr2.every(compare);        // true

  • some() 用于检测数组所有元素是否至少有1项符合某个指定函数的条件。 该方法不改变原始数组
    若使用空数组，此方法在一切情况下都返回 false。

    some() 方法使用指定函数检测数组中的所有元素：

    • 该函数接收3个参数，当前值_currentVal(必须)，当前值索引_index(可选)，当前数组_array(可选)。
    • 如果有一个元素满足条件，则表达式返回true , 剩余的元素不会再执行检测。
    • 如果没有满足条件的元素，则返回false。
    // 检测数组是否有元素大于10
function compare(val){
return val > 10;
}

let arr1 = [1, 11, 2, 22];
let arr2 = [2, 3, 4, 5];

arr1.some(compare);        // true
arr2.some(compare);        // false

  • filter() (过滤)用于找出数组中所有符合某个指定函数条件的元素、组成新的数组，并返回过滤后的新数组。 该方法不改变原始数组

    filter() 方法循环数组中的所有元素调用指定函数进行过滤：

    • 该函数接收3个参数，当前值_currentVal(必须)，当前值索引_index(可选)，当前数组_array(可选)。
    • 如果没有任何满足条件的元素，则返回空数组。
    // 找出大于10的元素
function compare(val){
return val > 10;
}

let arr1 = [1, 11, 2, 22];
let arr2 = [2, 3, 4, 5];

arr1.filter(compare);        // [11, 22]
arr2.filter(compare);        // []

  • forEach() (循环)用于对数组中的每个元素调用一次某个指定函数，没有返回值。
    除了抛出异常以外，没有办法中止或跳出 forEach() 循环。

    forEach() 方法接收一个指定函数：

    • 该函数接收3个参数，当前值_currentVal(必须)，当前值索引_index(可选)，当前数组_array(可选)。
    let arr = [1, 2, 3];

arr.forEach(function(item, index, arr){
arr[index] = val + 1;
});

console.log(arr);       // [2, 3, 4]

  • map() 将数组中的每一项调用某个指定函数后返回的结果组成新的数组，并返回该数组。 该方法不改变原始数组

    map() 方法接收一个指定函数：

    • 该函数接收3个参数，当前值_currentVal(必须)，当前值索引_index(可选)，当前数组_array(可选)。
    // 将数组中每一项乘以2
let arr = [1, 2 ,3];

arr.map(function(item){
return item * 2;
});

console.log(arr);       // [2, 4, 6]

  • reduce() 将数组中的每个元素调用某个指定的reducer函数，并把上次回调函数的返回值放在一个暂存器中传给下次回调函数，并返回最后一次回调函数的返回值。 该方法不改变原始数组

    reduce() 方法接收1个reducer函数和1个initialValue_初始值:

    • reducer函数接收4个参数，累计器_accumulator(必须)，当前值_currentVal(必须)，当前值索引_index(可选)，当前数组_array(可选)。
    • 如果没有提供初始值，则起始索引号为1，否则从索引0起始。
    • 如果数组为空且没有提供初始值，调用reduce将报错。
    // 累加方法
function add(prev, cur){
return prev + cur;
};
// 未传入初始值
[0, 1, 2, 3, 4].reduce(add);        // 10
// 传入初始值
[0, 1, 2, 3, 4].reduce(add, 5);     // 15

// 注意点：
// 1.如果数组仅有一个元素（无论位置如何）并且没有提供初始值，那么此唯一值将被返回并且callback不会被执行。
[ , 10].reduce(add);        // 10
// 2.如果有提供初始值但是数组为空，那么此唯一值将被返回并且callback不会被执行。
[].reduce(add, 11);         // 11
// 3.如果数组为空且没有提供初始值，会抛出TypeError 。
[].reduce(add);             // TypeError

// 例子：计算数组中每个元素出现的次数
var names = ['Alice', 'Bob', 'Tiff', 'Bruce', 'Alice'];

var countedNames = names.reduce(function (allNames, name) { 
if (name in allNames) {
    allNames[name]++;
}
else {
    allNames[name] = 1;
}
return allNames;
}, {});

console.log(countedNames);      // { 'Alice': 2, 'Bob': 1, 'Tiff': 1, 'Bruce': 1 }

  • reduceRight() 该方法的功能和 reduce() 功能是一样的，不同的是 reduceRight() 从数组的末尾向前循环每一项。

    //reduce 与 reduceRight 之间的区别
function add(prev, cur) { return prev + cur; }

let arr = ['1', '2', '3', '4', '5']; 

let left  = arr.reduce(add); 
let right = arr.reduceRight(add); 

console.log(left);      // "12345"
console.log(right);     // "54321"
